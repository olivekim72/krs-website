# 🌹 행사(이벤트) 추가하는 방법

이 홈페이지 시안은 **행사 목록 파일에 한 줄만 추가하면 카드가 자동으로 생성**됩니다.
별도 프로그램이나 코딩 없이, `data/events.json` 파일만 편집하면 됩니다.

## 추가 방법

`data/events.json` 파일을 열고, 아래 형식으로 항목 하나를 추가하세요:

```json
{
  "title": "행사 제목을 여기에",
  "date": "2026-08-15",
  "category": "전시회",
  "emoji": "🌹",
  "image": "images/내사진.jpg"
}
```

| 항목 | 설명 | 필수 |
|------|------|:---:|
| title | 행사 제목 (타이틀 글) | ✅ |
| date | 행사 날짜 (YYYY-MM-DD) | ✅ |
| category | 분류 (전시회 / 월간모임 / 교육 / 국제교류 등) | 선택 |
| image | 행사 이미지 경로. 없으면 emoji 표시 | 선택 |
| emoji | 이미지가 없을 때 보여줄 그림문자 | 선택 |

## 💡 더 편하게 하려면

지금은 파일을 직접 편집하는 "정적 사이트" 방식입니다.
**홈페이지에서 로그인 → 이미지 올리고 제목만 입력하면 즉시 게시되는** 관리자 페이지(CMS) 방식으로
업그레이드할 수 있습니다. (이 경우 웹 서버가 필요합니다 — 클로드에게 말씀해 주세요.)
